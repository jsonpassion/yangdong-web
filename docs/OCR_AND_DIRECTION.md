## Design Direction

- Accent color: Vision blue `#78d6f4` with heritage gold `#d8ad5b`, celadon green `#96c5ad`, and dancheong vermillion `#c86a50`.
- Tone: liquid glass panels over a dark heritage landscape, avoiding a single-color tech palette.
- Flow: hero -> heritage/context -> scrollytelling agenda -> challenge tuner -> results -> application -> asset request list.
- Visual system: spatial rings, contour lines, hanok roof silhouettes, shared-gallery frames, and interactive particles.
- Application: static SPA-safe form with local temporary storage; replace `APPLICATION_ENDPOINT` in `main.js` to submit to a real backend.
